**API Endpoints (`backend/app/api/v1/endpoints/developer_tools.py`):**
- `/developer-tools/setup` - Configure GitHub or Jira integration
- `/developer-tools/oauth/initiate` - Start OAuth flow for developer tools access
- `/developer-tools/oauth/complete` - Complete OAuth and store encrypted tokens
- `/developer-tools/github/repositories` - Fetch GitHub repositories with metadata
- `/developer-tools/github/pull-requests` - Fetch PRs with review status and metrics
- `/developer-tools/jira/issues` - Fetch Jira issues with status and progress info
- `/developer-tools/metrics/analyze` - Analyze development metrics and team performance
- `/developer-tools/webhooks/setup` - Configure real-time event notifications
- `/developer-tools/webhooks/events` - Process webhook events from GitHub/Jira
- `/developer-tools/status` - Get integration status and configuration details
